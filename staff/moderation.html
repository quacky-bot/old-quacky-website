Quacky Bot Staff Moderation

Color Key

Instructions for Helpers will be in this Color.

Instructions for Support Team Members (or Mods if applicable) will be in this Color.

Instructions for Moderators will be in this Color.

Instructions for Everyone will be in this Color.

After Reporting to a Mod/Warning

Check the user's warnings. Depending on the number of warnings they have, different moderation actions will be taken.

If you're a Helper, notify a Moderator if they need to take additional moderation actions.

If you're a Support Team Member and need to ban someone, ask an moderator.

If you're a Moderator, take the neccesary actions to moderate the user.

Moderation Actions with NSFW Warnings:

  • 2 NSFW Warns: Tempban for 1 Month
  • 3 NSFW Warnings: Permanent Ban

Moderation Actions with Overall Warnings:

  • 2 Warnings: Mute for 3 Days
  • 4 Warnings: Mute for 4 Days
  • 6 Warnings: Tempban for 1 Week
  • 7 Warnings: Permanent Ban

Spamming, Mass Mentioning, and Ghost Pinging

Warn the user then tell a moderator.

Purge the messages from the user then warn and mute the user for 1 day.

Improper Channel

Warn the user and delete the message.

Advertising

Warn the user then ask a moderator to delete the message.

Warn the user then delete the message.

Cursing, NSFW, and Racial Slurs

Warn the user then tell a moderator.

Warn and mute the user for 1 day then delete the message.

DM Advertising

Mention the user saying this is not allowed then notify a support team member or above.

Kick the user then tell a mod to ban them.

Ban the user.

Raiding or Multiple Users Spamming

Notify a moderator or admin immediately!

Lockdown the channel. If the spam moves to a different channel, lockdown the server.

Staff Abuse

Tell an Admin Immediately!
